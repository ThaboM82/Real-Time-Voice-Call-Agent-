# Real-Time Voice Call Agent

## ?? Overview
The Real-Time Voice Call Agent is a Node.js-based system that integrates **speech-to-text (STT)**, **text-to-speech (TTS)**, and **telephony APIs** to enable live, interactive voice calls powered by AI. It supports real-time transcription, natural language responses, and seamless audio streaming.

## ?? Features
- Backend services: Twilio integration, STT/TTS modules, database persistence
- Frontend app: Next.js interface for managing calls, agents, and analytics
- Voice synthesis: ElevenLabs + Deepgram integration for natural speech
- Dockerized setup: Ready for containerized deployment
- Secure configuration: .gitignore hardened to exclude secrets and environment files

## ?? Tech Stack
- Backend: Node.js, Express, SQLite3
- Frontend: Next.js, TypeScript, React
- APIs: Twilio, Deepgram, ElevenLabs
- DevOps: Docker, GitHub Actions (secret scanning)
